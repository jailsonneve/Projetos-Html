Projetos HTML5

Este repositório reúne uma coleção de sites, layouts e aplicações web desenvolvidos com HTML5, CSS3 e JavaScript.

Os projetos têm como foco a criação de experiências modernas, interativas e responsivas. É ideal para quem deseja se inspirar ou explorar boas práticas em front-end.

Entre os destaques estão um portfólio pessoal, uma lista de tarefas com banco de dados Firestore, uma página de busca de filmes usando Tomcat com Docker, e um índice navegável de todos os projetos.
